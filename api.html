Overview of the py3dframe package#

Main classes#

The package py3dframe is composed of the following main classes:

  • py3dframe.Rotation class is used to manage 3D rotations (alias of scipy.spatial.transform.Rotation).

  • py3dframe.Frame class is used to represent 3D frames of reference.

  • py3dframe.FrameTree class is used to manage hierarchical relationships between multiple frames (easier than using py3dframe.Frame.parent attributes).

  • py3dframe.FrameTransform class is used to manage 3D transformations between frames.
