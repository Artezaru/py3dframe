py3dframe
==========

The package ``py3dframe`` is composed of the following functions, classes and modules:

- ``py3dframe.Frame`` class is used to manage 3D frames.
- ``py3dframe.FrameBinder`` class is used create a hierarchy of frames.
- ``py3dframe.BasisUtils`` class of static methods to perform operations on basis
